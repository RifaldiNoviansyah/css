Jumbotron menunjukkan kotak besar untuk memanggil perhatian ekstra ke beberapa konten atau informasi khusus.

Jumbotron ditampilkan sebagai kotak abu-abu dengan sudut membulat. Ini juga memperbesar ukuran font dari teks di dalamnya.

Tip: Di dalam jumbotron Anda dapat meletakkan hampir semua HTML yang valid, termasuk elemen / kelas Bootstrap lainnya.

Gunakan elemen < div > dengan kelas .jumbotron untuk membuat jumbotron:
